## IMPLEMENTATION NOTES

### For World Map Screen
Each soup appears on the world map at its cultural origin. Hovering shows:
1. Soup name and flag emoji
2. Cultural tagline (the bold subtitle)
3. First paragraph (truncated to ~80 words)
4. [Select This Civilization] button

### For Character Card UI
When player selects, full two-paragraph monologue displays with:
- Full character voice and personality
- Strategic philosophy through cultural lens
- Victory conditions hinted at through narrative
- Gameplay style telegraphed through metaphors

### Narrative Functions
- **Diegetic voice**: Each soup is a character with real personality
- **Gameplay telegraph**: Monologue hints at actual mechanics
- **World-building**: Game feels alive because NPCs have strong voices
- **Replayability**: Each civilization tells a different story
- **Cultural authenticity**: Stereotypes are celebrated, not mocked
